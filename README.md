A 2 Player Fighting Game
Game can be run on main.exe

Player 1 Controls:
W: Jump
A: Left
D: Right
R: Attack1
T: Attack2

Player 2 Controls:
UP ARROW: Jump
LEFT ARROW: Left
RIGHT ARROW: Right
1 KEYPAD: Attack1
2 KEYPAD: Attack2
